Назначение

Гравитационные паллетные стеллажи (роликовые стеллажи) это система хранения высокой плотности. Данные стеллажи обеспечивают практически самый высокий коэффициент использования складского пространства. Использование таких стеллажей идеально для установки в складах с высокой оборачиваемостью груза, с такими стеллажами очень удобно контролировать сроки хранения груза.

Конструкция

Конструкция гравитационных стеллажей состоит из статичных элементов (рамы,балки) образующих каркас, и роликовых дорожек, которые укладываются на несущие балки под небольшим углом, по которым и перемещаются паллеты. Дорожки оснащены тормозными роликами, с помощью которых регулируется скорость перемещения паллеты с грузом.

Особенности применения

Гравитационные паллетные стеллажи подразумевают работу по системе FIFO (первый загружен, первый выгружен, таким образом зона погрузки и разгрузки четко разделены. Технически работа с такими стеллажами выглядит так, со стороны загрузки ставится поддон с грузом, по инерции он катится по роликовым направляющим по направлению к зоне выгрузки.

  • Преимущества
    • высокий коэффициент использования складского пространства
    • большая скорость обработки грузов
    • простота контроля грузопотоков
    • минимум персонала и техники
    • снижение рисков повреждения стеллажей
    • экономия электроэнергии на складе
    • работа по принципу FIFO
    • достаточно низкий срок окупаемости
    Недостатки
    • относительно высокая стоимость
    • повышенные требования к упаковке груза
