Назначение

Мобильные полочные или, как еще называют эту систему, передвижные стеллажи, предназначены для обеспечения максимального использования полезного пространства помещения под хранение. Данная система хранения позволяет увеличить эффективность помещения до 80%, за счет значительного сокращения количества проходов между стеллажами.

Конструкция

Данная конструкция идентична обычным полочным стеллажам, т.е. она также состоит из стоек на которые закреплены полочные уровни, количество уровней хранения может варьироваться, сами стеллажи размещаются на мобильные основания, которые перемещаются по специальным направляющим. Перемещение возможно с помощью механического или электрического привода.

Особенности применения

Мобильные полочные стеллажи размещаются одним модулем с единственным проходом между стеллажами. Для доступа к определенной секции хранения, оператор перемещает стеллажи, открывая нужный проход. Как и обычные полочные стеллажи, мобильная система хранения может быть оснащена различными аксессуарами ( полочными разделителями, балками для хранения шин, торцевыми или задними перегородками и т.д.).
